プロジェクト概要

Chronos Zeroは、就活作品として制作した3Dスタイリッシュアクションゲームです。 C++とDirectXを使用してImGuiを使用して開発ツールを作成し、 様々なシェーダー技術とアクションゲームに必要な要素を実装しました。 プレイヤーは制限時間内に敵を倒し、スコアを競います。

開発期間

4ヶ月

開発体制

個人制作

主要技術

C++ DirectX HLSL ImGui

技術的な実装

HLSL シェーダープログラミング

ゲームの視覚的な魅力を高めるために、高度なHLSLシェーダー技術を実装しました。 放射ブラー、画面ノイズ、カスケードシャドウマップ、PBRライティングなど、 多層的なエフェクトでリアルなゲーム体験を実現しています。

放射ブラー

画面中心から外側に向かってブラー効果を適用し、スピード感のある演出を実現

アルゴリズム解説
1

中心点からの距離を計算し、距離に応じたブラー強度を決定

2

中心方向に向かって複数のサンプル点を設定

3

各サンプル点の色を取得して加算

4

サンプル数で平均化して最終色を出力

HLSL実装コード
float3 color = float3(0.0f, 0.0f, 0.0f);
float2 uv = In.UV - g_uvOffset;
// ピクセルが中心から遠いほど大きく伸ばす
float factor = g_strength / g_sampleCount * length(uv);

// 現在のピクセルからブラー中心方向へ少しずつ寄った複数地点をサンプリング
for(int i = 0; i < g_sampleCount; i++)
{
    // サンプル数が増えるほどオフセットが大きくなる
    float offset = 1 - factor * i;
    color += g_inputTex.Sample(g_ss, uv * offset + g_uvOffset).rgb;
}

// 色を戻すために平均化して返す
return float4(color, 1) / g_sampleCount;
パラメータ説明
g_uvOffset ブラーの中心位置
g_strength ブラー効果の強度
g_sampleCount サンプリング回数（品質）

画面ノイズ

ダメージ演出やエフェクト強化のための太い縦帯ノイズ生成システム

アルゴリズム解説
1

画面を128分割して太い縦帯を作成

2

各帯ごとに時間ベースのランダムノイズを生成

3

UVオフセットでピクセル位置を水平方向にずらす

4

ノイズ値を色に加算して画面の乱れを表現

HLSL実装コード
// 太い縦ノイズ帯を生成
float verticalBandNoise(float2 uv, float time)
{
    // 画面を128分割（太さ調整）
    float band = floor(uv.y * 128.0);
    // 各帯ごとに同じノイズ値
    float noise = frac(sin(band + time * 5.0) * 43758.5453) * 2.0 - 1.0;
    return noise * g_noiseStrength * 0.02;
}

// メイン処理
float2 uv = In.UV;
if (g_enableNoise == 1)
{
    // UV座標を水平方向にずらす
    uv.x += verticalBandNoise(uv, g_time);
}

float3 color = g_inputTex.Sample(g_ss, uv).rgb;

if (g_enableNoise == 1)
{
    // ノイズを色に加算
    float n = verticalBandNoise(uv, g_time) * 10.0;
    color = saturate(color + n);
}
パラメータ説明
g_noiseStrength ノイズの強度調整
g_time 時間ベースのアニメーション
128.0 縦帯の分割数（太さ調整）
43758.5453 ランダム生成用のマジックナンバー

カスケードシャドウマップ

距離に応じて解像度を調整した高品質な影表現システム

アルゴリズム解説
1

視錐台を距離に応じて複数の領域に分割

2

各領域に最適化されたシャドウマップを生成

3

ピクセルの深度値に応じて適切なシャドウマップを選択

4

複数シャドウマップの境界で補間して滑らかに接続

HLSL実装コード
// カスケード選択
int SelectCascade(float depth)
{
    for(int i = 0; i < CASCADE_COUNT - 1; i++)
    {
        if(depth < g_cascadeDistances[i])
            return i;
    }
    return CASCADE_COUNT - 1;
}

// シャドウ計算
float CalculateShadow(float3 worldPos, float depth)
{
    int cascade = SelectCascade(depth);
    
    // 選択されたカスケードの座標変換
    float4 shadowPos = mul(float4(worldPos, 1.0), g_shadowMatrices[cascade]);
    shadowPos /= shadowPos.w;
    
    // シャドウマップからサンプリング
    float shadowDepth = g_shadowMaps[cascade].Sample(g_shadowSampler, shadowPos.xy).r;
    
    // 影判定
    return shadowPos.z > shadowDepth + SHADOW_BIAS ? 0.0 : 1.0;
}
パラメータ説明
CASCADE_COUNT 分割数（通常4分割）
g_cascadeDistances 各カスケードの距離境界
SHADOW_BIAS シャドウアクネ防止バイアス
複数のシャドウマップを距離別に使い分け
近距離は高解像度、遠距離は低解像度で最適化
品質とパフォーマンスのバランスを実現

PBRベースライティング

物理ベースレンダリングによるリアルな光の表現

アルゴリズム解説
1

Cook-Torranceモデルによる鏡面反射計算

2

Lambert拡散反射とフレネル項の組み合わせ

3

金属度パラメータによる反射特性の調整

4

粗さパラメータによる表面の微細構造表現

HLSL実装コード
// フレネル項（Schlick近似）
float3 FresnelSchlick(float cosTheta, float3 F0)
{
    return F0 + (1.0 - F0) * pow(1.0 - cosTheta, 5.0);
}

// 法線分布関数（GGX）
float DistributionGGX(float NdotH, float roughness)
{
    float a = roughness * roughness;
    float a2 = a * a;
    float NdotH2 = NdotH * NdotH;
    
    float num = a2;
    float denom = (NdotH2 * (a2 - 1.0) + 1.0);
    denom = PI * denom * denom;
    
    return num / denom;
}

// PBRライティング計算
float3 CalculatePBR(float3 albedo, float metallic, float roughness,
                    float3 normal, float3 viewDir, float3 lightDir)
{
    float3 F0 = lerp(float3(0.04, 0.04, 0.04), albedo, metallic);
    float3 halfwayDir = normalize(lightDir + viewDir);
    
    float NdotV = max(dot(normal, viewDir), 0.0);
    float NdotL = max(dot(normal, lightDir), 0.0);
    float NdotH = max(dot(normal, halfwayDir), 0.0);
    
    // Cook-Torrance BRDF
    float3 F = FresnelSchlick(max(dot(halfwayDir, viewDir), 0.0), F0);
    float D = DistributionGGX(NdotH, roughness);
    float G = GeometrySmith(normal, viewDir, lightDir, roughness);
    
    float3 numerator = D * G * F;
    float denominator = 4.0 * NdotV * NdotL + 0.001;
    float3 specular = numerator / denominator;
    
    float3 kS = F;
    float3 kD = float3(1.0, 1.0, 1.0) - kS;
    kD *= 1.0 - metallic;
    
    return (kD * albedo / PI + specular) * radiance * NdotL;
}
パラメータ説明
albedo 材質の基本色
metallic 金属度（0.0-1.0）
roughness 表面の粗さ（0.0-1.0）
F0 垂直入射時の反射率
物理的に正確な光の散乱計算
マテリアルの金属度・粗さパラメータ
リアルタイムでの高品質レンダリング
